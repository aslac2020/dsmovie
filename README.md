# Projeto DSmovie

# Sobre o projeto

Projeto Semana React Spring Boot - Nelio Alves

Consiste em um sistema de filmes, onde é registrado as avaliações como nome, imagem, de alguns filmes, e o mesmo você pode
atualizar avaliações.
